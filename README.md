# Notes Application

# A simple and responsive Notes Application built using React.js and Tailwind CSS.  
# This application allows users to add, view, and delete notes easily with a clean and modern user interface.

---

#  Features

- 1. Add Notes
- 2. Delete Notes
- 3. Responsive Design
- 4. Modern UI with Tailwind CSS
- 5. Dynamic Rendering using React Hooks
- 6. Navbar and Footer Included
- 7. User-Friendly Interface
- 8. Real-time Note Updates

---

# Technologies Used

- 1. React.js
- 2. Tailwind CSS
- 3. JavaScript
- 4. HTML5
- 5. CSS3

# ⚙️ Installation & Setup

## 1. Clone the Repository

```bash
git clone <your-repository-link>
```

## 2️. Navigate to Project Folder

```bash
cd notes-application
```

## 3️. Install Dependencies

```bash
npm install
```

## 4️. Start Development Server

```bash
npm run dev
```
